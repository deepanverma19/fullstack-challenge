Welcome to my web application

This web application allow the web UI user to communicate with non web UI user. It allows the web user to create and read individual contacts where the contact name and method of communication are stored which is ideally E-Mail. The web UI allows web user to communicate with the existing contacts using conversations where the conversations are communication chain centered around a subject. Also the contact can respond on the communication chain and the message appears in the correct conversation.
